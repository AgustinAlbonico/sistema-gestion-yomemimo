# Sistema de Gestión — Monorepo

Estructura inicial del monorepo preparada con Turborepo y pnpm workspaces.

Comandos básicos:

- Instalar dependencias (requiere `pnpm`):

```powershell
pnpm install -w
```

- Arrancar en desarrollo (usa `turbo`):

```powershell
pnpm run dev
```

Para más información, revisar `docs/DOCKER-SETUP-COMPLETO.md` y `docs/stack-tecnologico.md`.
