# Improved Prompt

# Objective
Execute all tests in the monorepo and diagnose any failures

## Scope
- Backend unit tests (NestJS/Jest)
- Frontend unit tests (React/Vitest)
- E2E tests (Playwright)

## Steps
1. Run backend tests: `npm run test` in `apps/backend/`
2. Run frontend tests: `npm run test` in `apps/frontend/`
3. Run E2E tests: `npm run test:e2e` in `apps/frontend/`

## Analysis Required
For each failing test:
- Identify the root cause
- Provide the specific error message
- Suggest the fix needed
- Categorize failures (assertion error, timeout, missing dependency, etc.)

## Expected Output
- Summary of test results (passed/failed totals per suite)
- Detailed report of failures with root causes
- Prioritized list of fixes needed
